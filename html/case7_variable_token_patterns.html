Case 7 — Token đổi pattern theo phiên bản

Mục tiêu: viết Regex đủ “mềm dẻo” để match nhiều kiểu token.

  • v1: TK-12345-ABCDE
  • v2: TOKEN{A1B2C3D4E5}
  • v3: [TKN] SID_2025_10_18_0099
  • v4: TKN-xyz_987-ALPHA.beta
Expect matches (4):
TK-12345-ABCDE
TOKEN{A1B2C3D4E5}
SID_2025_10_18_0099
TKN-xyz_987-ALPHA.beta
