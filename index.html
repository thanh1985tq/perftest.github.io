JMeter Correlation Lab (Static)

Bộ bài thực hành xử lý tham số động (Correlation) với nhiều mức độ khó và định dạng response khác nhau. Phục vụ trích xuất bằng Regular Expression Extractor và JSON Path Extractor.

Gợi ý dùng trong JMeter: chạy một web server tĩnh trỏ vào thư mục này, ví dụ: python -m http.server 8080 -d jmeter-correlation-lab. Sau đó cấu hình HTTP Request Sampler tới các đường dẫn trên để thực hành trích xuất và dùng lại biến.
